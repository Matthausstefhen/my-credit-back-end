# MyCredit
Projeto da cadeira de **Web Services**

### Descrição

> Carteira digital
> Back End

### Tecnologias 

* Express

## Instalação

``` bash
# dev
$ yarn dev
````  
````bash
# start
$ yarn start
````  
````bash
# build
$ yarn build
````

```` bash
# prodution
$ yarn prodution
````
